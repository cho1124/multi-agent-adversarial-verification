# 02. 외부 의존성 제거

## 문제 (R1 C-04, R2 C-17, R8 C8-5)

3개 서드파티 패키지(rembg/pydantic/keyring)가 Blender 번들 Python과 충돌하거나 미포함.

## 최종 해소

서드파티 패키지 0개. 모든 기능을 표준 라이브러리 + Blender 빌트인으로:
- 배경 제거: remove.bg API (HTTP 호출)
- 스키마 검증: 순수 dict + isinstance
- API 키: AddonPreferences → userpref.blend
- EXIF 제거: 순수 바이트 조작
